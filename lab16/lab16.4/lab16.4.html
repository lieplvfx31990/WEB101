Lab 16.4

Câu lệnh switch

  1. Sử dụng câu lệnh switch để ghi string sau cho 'language':
  2. Chinese or Mandarin: 'MOST number of native speakers!'
  3. Spanish: '2nd place in number of native speakers'
  4. English: '3rd place'
  5. Hindi: 'Number 4'
  6. Arabic: '5th most spoken language'
  7. Tất cả các log đơn giản khác: 'Great language too :D'

Thực hành

Using switch statement

let language = "";
let retval;    // The variable that will hold the message about language

switch (language) {
    case "Chinese":
    case "Mandarin":
        retval = "MOST number of native speakers!";
        break;
    case "Spanish":
        retval = "2nd place in number of native speakers";
        break;
    case "English":
        retval = "3rd place";
        break;
    case "Hindi":
        retval = "Number 4";
        break;
    case "Arabic":
        retval = "5th most spoken language";
        break;
    default:
        retval = "Great language too :D";
}

// Print the message about considered language to the console
console.log(retval);
