Bài toán tổng hợp #2

Lab 16.6.1. Đội nào chiến thắng? (phần 1)

Có hai đội thể dục dụng cụ là Dolphins và Koallas. Họ thi đấu với nhau 3 lần. Đội nào có điểm trung bình cao nhất sẽ chiến thắng!

  1. Nhiệm vụ của bạn:
    • Tính điểm cho từng đội, sử dụng dữ liệu kiểm tra bên dưới.
    • So sánh điểm trung bình của hai đội để tìm ra đội chiến thắng và in ra console. Đừng quên là có thể hòa, nên hãy kiểm tra điều đó (hòa tức là họ có điểm trung bình giống nhau)
    • Bonus 1: Chúng ta được thêm một quy tắc như sau: số điểm tối thiểu là 100 điểm. Theo quy tắc này, một đội sẽ dành chiến thắng chỉ khi họ có điểm trung bình cao hơn đội còn lại, và đồng thời có ít nhất 100 điểm. Gợi ý: Sử dụng toán tử logic để kiểm tra điểm số tối thiểu cũng như các else-if block!
    • Bonus 2:Điểm số tối thiểu cũng áp dụng cho việc hòa trận!
      Trường hợp hòa nhau chỉ xảy ra khi cả hai đội có số điểm giống nhau mà phải lớn hơn hoặc bằng 100 điểm. Nếu không thì không có đội nào giành chiến thắng cả.
  2. Dữ liệu kiểm tra:
    • Dữ liệu 1: Dolphins được 96, 108 và 89 điểm. Koalas được 88, 91 và 110 điểm
    • Dữ liệu Bonus 1: Dolphins được 97, 112 và 101 điểm. Koalas được 109, 95 và 123 điểm
    • Dữ liệu Bonus 2: Dolphins được 97, 112 và 101 điểm. Koalas được 109, 95 và 106 điểm

Thực hành

Phần cơ bản
Tìm đội chiến thắng dựa vào điểm trung bình.

const dolp_points = [96, 108, 98];     // The array contains Dolpins' points
const koal_points = [88, 91, 110];     // The array contains Koalas' points
let dLen = dolp_points.length;
let kLen = koal_points.length;
let dAve = 0;    // The variable contains the average point of Dolphins
let kAve = 0;    // The variable contains the average point of Dolphins
let msg = "";    // The variable contains the message that will be printed out the console.

// Update value for dAve variable
for (let i = 0; i < dLen; i++) {
    dAve += dolp_points[i];
}
dAve /= dLen;

// Update value for kAve variable
for (let i = 0; i < kLen; i++) {
    kAve += koal_points[i];
}
kAve /= kLen;

// Specify which team won or they drew with each other
if (dAve < kAve) {
    msg = "Đội Koalas thắng";
} else if (dAve == kAve) {
    msg = "Hai đội hòa nhau";
} else {
    msg = "Đội Dolphins thắng";
}

// Show the value of msg out the console.
console.log(msg);

 


Bonus 1
Tìm đội chiến thắng dựa vào điểm trung bình tối thiểu là 100 điểm.

const dolp_points = [97, 112, 101];     // The array contains Dolpins' points
const koal_points = [109, 95, 123];     // The array contains Koalas' points
const minAvePoint = 100;    // The minimum average point that each of teams had to earn.
let dLen = dolp_points.length;
let kLen = koal_points.length;
let dAve = 0;    // The variable contains the average point of Dolphins
let kAve = 0;    // The variable contains the average point of Dolphins
let msg = "";    // The variable contains the message that will be printed out the console.

// Update value for dAve variable
for (let i = 0; i < dLen; i++) {
    dAve += dolp_points[i];
}
dAve /= dLen;

// Update value for kAve variable
for (let i = 0; i < kLen; i++) {
    kAve += koal_points[i];
}
kAve /= kLen;

// Specify which team won or they drew with each other
if ((dAve < kAve) && (kAve >= minAvePoint)) {
    msg = "Đội Koalas thắng";
} else if (dAve == kAve) {
    msg = "Hai đội hòa nhau";
} else if ((dAve > kAve) && (dAve >= minAvePoint)) {
    msg = "Đội Dolphins thắng";
} else {
    msg = "Không rõ kết quả thế nào";
}

// Show the value of msg out the console.
console.log(msg);
