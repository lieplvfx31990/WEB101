Hàm trong hàm

  1. Tạo một hàm là 'describePopulation'. Sử dụng kiểu hàm mà bạn muốn nhất. Hàm này nhận hai đối số là 'country' và 'population', và trả về string như sau: 'China has 1441 million people, which is about 18.2% of the world'.
  2. Để tính phần trăm 'describePopulation', hãy gọi 'percentageOfWorld1' mà bạn đã tạo ra trước đó.
  3. Gọi 'describePopulation' với dữ liệu cho 3 nước bất kỳ.

Thực hành

Tạo hàm

function percentageOfWorld1(population) {
    // The returned number is rounded to 2 digits in the decimal part.
    return ((population / 7900000000) * 100).toFixed(2);
}

function describePopulation(country, population) {
    return `${country} has ${population / 1000000} million people, which is about ${percentageOfWorld1(population)}% of the world`;
}

// China has 1425.49 million of people.
let chn = describePopulation("China", 1425.49 * 1000000);

// United State has 340.91 million of people.
let usa = describePopulation("United State", 340.91 * 1000000);

// Japan has 122.96 million of people.
let jap = describePopulation("Japan", 122.96 * 1000000);

// Print the results to the console.
console.log(chn);
console.log(usa);
console.log(jap);
