Lab 18.1

Giới thiệu về Object (5 phút)

  • Tạo một object là 'myCountry' cho quốc gia bạn chọn, chứa các thuộc tính 'country', 'capital', 'language', 'population' và 'neighbours' (aray như chúng ta đã thực hành trong bài trước).

Thực hành

Create an Object

// The Object contains brief information about my country
const myCountry = {
    country: "Vietnam",
    capital: "Hanoi",
    language: "Vietnamese",
    population: 100.3,    // in million
    neighbours: ["China", "Laos", "Cambodia"]
}
