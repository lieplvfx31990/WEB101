Lab 18.2

Dấu chấm với dấu ngoặc (10 phút)

  1. Sử dụng object từ bài tập trước, in string sau ra console: 'Finland has 6 million finnish-speaking people, 3 neighbours countries and a capital called Helsinki.'
  2. Tăng dân số của của đất nước thêm hai triệu người bằng cách sử dụng dấu chấm (object.population) , và sau đó giảm đi hai triệu người bằng dấu ngoặc (object['population']) để truy cập vào các thuộc tính.

Thực hành

Using dot (.) and square brackets ([]) for accessing JavaScript Object's properties / methods

// The Object contains brief information about my country
const myCountry = {
    country: "Vietnam",
    capital: "Hanoi",
    language: "Vietnamese",
    population: 100.3,    // in million
    neighbours: ["China", "Laos", "Cambodia"]
}

// Prints required message to the console
let oMsg = `${myCountry.country} has ${myCountry.population} million ${myCountry.language}-speaking people, ${myCountry.neighbours.length} neighbours countries and a capital called ${myCountry.capital}.`;
console.log(oMsg);

// Increases the population by 2 (million).
myCountry.population += 2;
// Prints out the population property of myCountry to check the result
console.log(myCountry['population']);

// Decreases the population by 2 (million).
myCountry['population'] -= 2;
// Prints out the population property of myCountry to check the result
console.log(myCountry.population);
