Vòng lặp ngược và vòng lặp trong vòng lặp (15 phút)

  1. Lưu trử array của những array này vào biến 'listOfNeighbours' [['Canada', 'Mexico'], ['Spain'], ['Norway', 'Sweden', 'Russia']];
  2. Nếu như các mảng con có ít nhất hai phần tử, in ra lần lượt phần tử từ thứ 2 trở đi. Ví dụ với dữ liệu như trên sẽ in ra.
    Sweden
    Mexico
    Russia
  3. Bạn sẽ cần một vòng lặp bên trong vòng lặp cho điều này. Điều này thực sự hơi phức tạp, vì vậy đừng lo lắng nếu nó quá khó đối với bạn! Bạn sẽ giải quyết được vấn đề này.

Thực hành

Reverse for loop for array countdown, and nested for loop in JavaScript

// Declare array of listOfNeighbours
const listOfNeighbours = [
  ['Canada', 'Mexico'],
  ['Spain'],
  ['Norway', 'Sweden', 'Russia']
];

// For all sub-arrays of listOfNeighbours, check if each of them
// has at least 2 elements; If it's true, print out their elements
// beginning from the second ones of each sub-array, respectively.
// Perform the process in the direction from the last sub-array to the
// first sub-array of listOfNeighbours
let maxLen = 0;
let selectedNeighbours = [];  // The array holds elements that will be printed out
for (let j = 1; ; j++) {
    for (let i = listOfNeighbours.length - 1; i >= 0; i--) {
        let subArrLen = listOfNeighbours[i].length;
        maxLen = maxLen < subArrLen ? subArrLen : maxLen;
        if (subArrLen > j) {
            selectedNeighbours.push(listOfNeighbours[i][j]);
        } else {
            continue;
        }
    }
    if (j > maxLen) break;
}

// Print the elements of selectedNeighbours line by line
for (let cnt = 0; cnt < selectedNeighbours.length; cnt++) {
    console.log(selectedNeighbours[cnt]);
}
