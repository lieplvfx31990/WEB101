String và Template Literal

  1. Tạo lại biến 'description' từ lần gán trước, lần này hãy sử dụng cú pháp template literal.

Thực hành

Create string using template literal

// Declare some variables
const country = "Vietnam";
const continent = "Asia";
const isIsland = false;
const language = "Vietnamese";
let population = 100.3;  // In million

// Perform division operator
console.log(population / 2);

// Perform increment operator
population++;
console.log(population);

// Declare a variable that holds the value of Finland's population
let fin_population = 6;  // in million
// Perform comparison operator (>)
console.log(population > fin_population);

// Declare a variable that holds the average population of the World
let average_population = 33;  // in million
// Perform comparison operator (<)
console.log(population < average_population);

// Create string using template literal
let description = `${country} and its ${population} million people speak ${language}`;
console.log(description);
