Lab 15.6.1

So Sánh Chỉ Số BMI (phần 1)

Mark và John đang cố so sánh chỉ số BMI (Body Mass Index) của họ, được tính bằng công thức: BMI = mass/[(height)^2] = mass/(height*height) (mass là khối lượng của cơ thể tính bằng kg và height là chiều cao của cơ thể tính bằng mét).

  1. Nhiệm vụ của bạn:
    • Lưu khối lượng và chiều cao của Mark và John vào các biến.
    • Tính chỉ số BMI của John và Mark theo công thức (bạn có thể tính cả hai).
    • Tạo biến Boolean 'markHeigherBMI' chứa thông tin về việc liệu chỉ số BMI của Mark có cao hơn BMI của John không.
  2. Dữ liệu kiểm tra:
    • Dữ liệu 1: Mark nặng 78kg và cao 1.69m. John nặng 92kg và cao 1.95m.
    • Dữ liệu 2: Mark nặng 95kg và cao 1.88m. John nặng 85kg và cao 1.76m.

Thực hành (phần 1)

Part 1

let markHeight;  // Mark's height
let markMass;  // Mark's mass
let johnHeight;  // John's height
let johnMass;  // John's mass

let markBMI;  // Mark's body mass index
let johnBMI;  // John's body mass index
let markHigherBMI;  // Mark's BMI higher John's BMI or not

markMass = 78;
markHeight = 1.69;
johnMass = 92;
johnHeight = 1.96;

markBMI = markMass / markHeight ** 2;
johnBMI = johnMass / johnHeight ** 2;
markHigherBMI = markBMI > johnBMI;

console.log("DỮ LIỆU 1:");
console.log(`Mark's mass: ${markMass}`);
console.log(`Mark's height: ${markHeight}`);
console.log(`Mark's BMI: ${markBMI.toFixed(2)}`);
console.log(`John's mass: ${johnMass}`);
console.log(`John's height: ${johnHeight}`);
console.log(`John's BMI: ${johnBMI.toFixed(2)}`);
console.log(`markHigherBMI: ${markHigherBMI}`);

console.log("--------------------------------";

markMass = 95;
markHeight = 1.88;
johnMass = 85;
johnHeight = 1.76;

markBMI = markMass / markHeight ** 2;
johnBMI = johnMass / johnHeight ** 2;
markHigherBMI = markBMI > johnBMI;

console.log("DỮ LIỆU 2:");
console.log(`Mark's mass: ${markMass}`);
console.log(`Mark's height: ${markHeight}`);
console.log(`Mark's BMI: ${markBMI.toFixed(2)}`);
console.log(`John's mass: ${johnMass}`);
console.log(`John's height: ${johnHeight}`);
console.log(`John's BMI: ${johnBMI.toFixed(2)}`);
console.log(`markHigherBMI: ${markHigherBMI}`);

Lab 15.6.2

So Sánh Chỉ Số BMI (phần 2)

Sử dụng ví dụ về BMI từ Lab 15.6.1 và code bạn đã viết để cải thiện lại.

Nhiệm vụ của bạn:

  • In một output tốt ra console, hiển thị ai có chỉ số BMI cao hơn. Thông báo hiển thị sẽ là "Mark BMI is higher than John's!" hoặc "John's BMI is higher than Mark's!".
  • Sử dụng template literal bao gồm các giá trị BMI trong output. Ví dụ: "Mark's BMI (28.3) is higher than John's (23.9)!".

Gợi ý: Sử dụng câu lệnh if/else

Thực hành(phần 2)

Part 2

let markHeight;  // Mark's height
let markMass;  // Mark's mass
let johnHeight;  // John's height
let johnMass;  // John's mass

let markBMI;  // Mark's body mass index
let johnBMI;  // John's mass
let markHigherBMI;  // Mark's BMI higher John's BMI or not

markMass = 78;
markHeight = 1.69;
johnMass = 92;
johnHeight = 1.96;

markBMI = markMass / markHeight ** 2;
johnBMI = johnMass / johnHeight ** 2;
markHigherBMI = markBMI > johnBMI;

console.log("NHIỆM VỤ 1:");
if (markHigherBMI) {
    console.log("Mark's BMI is higher than John's!");
} else {
     console.log("John's BMI is higher than Marks's!");
}

console.log("------------------------------------");

console.log("NHIỆM VỤ 2:");
if (markHigherBMI) {
    console.log(`Mark's BMI (${markBMI.toFixed(2)}) is higher than John's (${johnBMI.toFixed(2)})!`);
} else {
    console.log(`John's BMI (${johnBMI.toFixed(2)}) is higher than Mark's (${markBMI.toFixed(2)})!`);
}
