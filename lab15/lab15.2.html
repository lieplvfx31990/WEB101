let, const và var

  1. Đặt giá trị của 'language' thành ngôn ngữ được nói ở nơi bạn sống (một số quốc gia có nhiều ngôn ngữ, nhưng chỉ cần chọn một). Ví dụ như 'English', 'Vietnamese'.
  2. Hãy nghĩ xem những biến nào có thể sử dụng const (những giá trị nào không bao giờ thay đổi?). Sau đó, chuyển các biến mà bạn đã nghĩ ra thành biến khai báo bằng const.
  3. Giờ hãy thử thay đổi giá trị một trong các biến trên và quan sát xem điều gì sẽ xảy ra.

Thực hành

Variables declaration with const, let, and var

// Declare some variables with 'let', 'const', and 'var'
const country = "Vietnam";
const continent = "Asia";
let population = 100.3;
const isIsland = false;
var language = "Vietnamese";

// Try to re-assign some declared variables, see what happens
try {
    isIsland = true;
    console.log(isIsland);
} catch(const_err) {
    console.log(const_err);
}

try {
    population = 101.4;
    console.log(population);
} catch(let_err) {
    console.log(let_err);
}

try {
    language = "English";
    console.log(language);
} catch(var_err) {
    console.log(var_err);
}
